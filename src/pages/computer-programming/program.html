Other Editor Stuff

Click the plus button to create a new file.
Click the "X" button to delete a file.
Double click on an editor tab to rename the file.


Editor Shortcuts

There are plenty of shortcuts that the editor supports, but here are the most commonly used ones.
Shortcut Action
CTRL + Z undo
CTRL + Y redo
CTRL + C copy
CTRL + X cut
CTRL + V paste
CTRL + A select all
CTRL + F search
CTRL + H search & replace
CTRL + S save
CTRL + ENTER run code
CTRL + / toggle line comment
TAB increase indent
SHIFT + TAB decrease indent
SHIFT + LEFT select left
SHIFT + RIGHT select right
SHIFT + UP select up
SHIFT + DOWN select down
CTRL + ALT + UP add multi-cursor above
CTRL + ALT + DOWN add multi-cursor below
ESC collape multi-cursors
ALT + UP move line up
ALT + DOWN move line down
